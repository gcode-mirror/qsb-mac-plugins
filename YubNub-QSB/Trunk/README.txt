# YubNub plugin for Google Quick Search Box

A [Google Quick Search Box][http://code.google.com/p/qsb-mac] plugin to send search commands to [YubNub][http://www.yubnub.org]

#DISCLAIMER

This product was NOT developed by the team at YubNub, or Google. They should not be held responsible if this breaks, or if you run into problems.

# INSTALL
- Download the latest version from the [Downloads section][http://www.github.com/gfontenot/YubNub-QSB/downloads]
- Mount the DMG
- Drag the plugin into your plugins folder
- Re-Start Google Quick Search Box

That's it!

# USAGE

- Invoke QSB
- Hit the space bar to begin a text item
- Type your command
- Hit tab to lock the text item in
- Press "y" to select "Search with YubNub"
- Hit enter

Your webbrowser will open and navigate to the chosen command

###Basic command examples (taken from yubnub.org):

- gim porsche 911 - Do a google Image Search for Porsche 911
- random 100 - Pick a random number between 1 and 100
- man random - Display help for the "random" command
- create - Create a new command
- ls - List all available commands
- xe -amount 100 -from USD -to EUR - Convert 100 Dollars to Euros with xe.com
